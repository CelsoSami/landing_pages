# Celso Sami - Portfolio

Personal portfolio website built with HTML, CSS, and JavaScript. Bilingual (EN/PT-BR), dark neon theme, fully responsive.

**Live:** [celsosami.github.io/landing_pages](https://celsosami.github.io/landing_pages/)

## Features

- Bilingual support (English / Portuguese)
- Dark neon theme (blue `#00d4ff` + beige `#e8d5b7`)
- Interactive particle system on home section
- Typed.js text animation
- Scroll-reveal animations (IntersectionObserver)
- Responsive hamburger menu for mobile
- Photo gallery with hover effects and scroll animation
- FAQ accordion with `<details>` elements
- Timeline-style experience section
- Profile photo with animated neon border

## Sections

| Page | Description |
|------|-------------|
| **Home** | Hero with typed animation, profile photo, social links, particles |
| **About (page)** | Full career narrative, timeline, education, skills, FAQ |
| **About (section)** | Quick summary with stats on main page |
| **Expertise** | 6 cards covering core competencies |
| **Projects** | Portfolio cards with hover overlay |
| **Contact** | Info cards (LinkedIn, GitHub, Email, Location) |
| **The Wife** | Photo gallery tribute page |
